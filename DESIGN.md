# Animus — Design System

## Вдохновение
Anthropic-style: тёплый минимализм, много воздуха, утончённость без вычурности. Ощущение ремесла и заботы, а не корпоративного шаблона.

## Принципы дизайна

1. **Тёплый, не холодный** — кремовые оттенки вместо чистого белого, терракот вместо синего
2. **Много воздуха** — секции с большими padding (7rem+), карточки с просторными отступами
3. **Тонкие границы** — бордеры 1px в тёплых оттенках, без тяжёлых теней
4. **Микро-анимации** — плавные reveal при скролле, hover эффекты, ничего резкого
5. **Чистая иконография** — SVG stroke-only, тонкие линии (stroke-width: 1.5-2)
6. **Нет AI-слопа** — никаких градиентных blob'ов, 3D-рендеров, стоковых иллюстраций
7. **Геометрические акценты** — тонкие линии, точки, минимальные декоративные элементы

## Компоненты

### Кнопки
- Primary: заливка accent, белый текст, subtle тень
- Ghost: бордер surface, dim текст, hover → accent бордер
- Hover: translateY(-2px), усиление тени
- border-radius: 8px

### Карточки
- Белый фон, бордер surface (1px)
- Hover: лёгкая тень, бордер accent
- border-radius: 12px
- Padding: 2rem

### Форма
- Инпуты: bg-alt фон, тонкий бордер, focus → accent бордер
- Скруглённые углы 8px
- Placeholder в dim цвете

## Адаптив
- Desktop: max-width 1200px, grid 2-3 колонки
- Tablet (768px): 1-2 колонки, бургер меню
- Mobile (480px): 1 колонка, кнопки full-width

## SEO / Техническое
- Семантический HTML5 (header, nav, section, footer)
- Open Graph мета-теги (ru_BY locale)
- geo.region BY, geo.placename Minsk
- Адаптивный viewport
- Чистые URL с якорями (#services, #contact)
